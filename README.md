# SkyGuardian

Новая основа панели управления SkyGuardian.

## Реализовано

- отдельная публичная страница технических работ;
- отдельная страница авторизации;
- адаптивная тёмная административная панель;
- разделы «Новости» и «Воздушная тревога» с одинаковыми формами и полностью раздельными данными;
- до 10 каналов данных в каждом разделе;
- спойлеры каналов, включение/отключение без удаления настроек;
- обязательные поля формы: название, источник, технический аккаунт, канал публикации, частота проверки и способ публикации;
- необязательные фильтры, исключающие слова и редактор дополнительного текста;
- отдельное управление техническими аккаунтами, до 10 аккаунтов;
- подтверждение удаления, уведомления, пустые состояния и страницы ошибок.

## Запуск

Требуется PHP 8.1 или новее.

```bash
mkdir -p storage
chmod 775 storage
SG_ADMIN_EMAIL=admin@example.com SG_ADMIN_PASSWORD='change-this-password' php -S 0.0.0.0:8080 -t public
```

Открыть: `http://SERVER_IP:8080`.

Данные хранятся в `storage/skyguardian.json`. Перед рабочим использованием задайте собственные `SG_ADMIN_EMAIL` и `SG_ADMIN_PASSWORD`.

## Структура

- `public/index.php` — маршрутизация, авторизация и серверная логика;
- `public/assets/app.css` — централизованный дизайн и адаптивность;
- `public/assets/app.js` — спойлеры, редактор, модальные окна и интерфейс;
- `storage/` — локальное хранилище данных.
